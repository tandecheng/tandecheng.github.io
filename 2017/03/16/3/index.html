班扎古鲁白玛的沉默

发表于 2017-03-16

《班扎古鲁白玛的沉默》

作者：扎西拉姆·多多

你见，或者不见我，

我就在那里，不悲不喜。

你念，或者不念我，

情就在那里，不来不去。

你爱，或者不爱我，

爱就在那里，不增不减。

你跟，或者不跟我，

我的手就在你手里，不舍不弃。

来我的怀里，或者，让我住进你的心里。

默然 相爱，寂静 欢喜。
